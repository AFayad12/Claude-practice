# HTML Review Skill

When I ask you to review an HTML file, always check these things in order:

1. Is the HTML structure correct? (html, head, body tags)
2. Are all links and buttons working?
3. Does it look good on mobile screen sizes?
4. Is the code clean and easy to read?
5. Are there any spelling mistakes in the visible text?

After checking, give me:
- A summary of what is good
- A list of things to fix
- The fixed code if anything needs changing
